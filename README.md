# Calculadora Windows Forms

Esta es una calculadora desarrollada en C# que combina funciones básicas y científicas.  
Es una calculadora un poco simple pero funcional.

# Características:
**Operaciones básicas:**  
  -Suma  
  -Resta  
  -Multiplicación  
  -División.  
**Funciones avanzadas:**   
  -Cálculo de seno (`sen`)  
  -Coseno (`cos`)  
  -Tangente(`tan`)  
  -Potencias(`n^x`)  
  -Logaritmo(`log`)  
  -Logaritmo natural(`ln`)  
**Interfaz gráfica:** Diseñada de forma intuitiva utilizando Windows Forms.
